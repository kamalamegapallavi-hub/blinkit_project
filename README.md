# 🛒 Blinkit Sales Analysis using Python

## 📌 Project Overview
This project analyzes Blinkit's retail sales dataset using Python to identify business insights and visualize sales performance. The analysis includes data cleaning, exploratory data analysis (EDA), KPI calculations, and multiple visualizations to support business decision-making.

---

## 📊 Dataset Information
- **Rows:** 8,523
- **Columns:** 12

### Features
- Item Fat Content
- Item Identifier
- Item Type
- Outlet Establishment Year
- Outlet Identifier
- Outlet Location Type
- Outlet Size
- Outlet Type
- Item Visibility
- Item Weight
- Sales
- Rating

---

## 🎯 Business Requirements
The project calculates the following KPIs:
- Total Sales
- Average Sales
- Number of Items Sold
- Average Customer Rating

---

## 📈 Visualizations
- Total Sales by Fat Content (Pie Chart)
- Total Sales by Item Type (Bar Chart)
- Fat Content by Outlet (Grouped Bar Chart)
- Total Sales by Outlet Establishment (Line Chart)
- Sales by Outlet Size (Pie Chart)
- Sales by Outlet Location (Horizontal Bar Chart)

---

## 🧹 Data Cleaning
- Standardized inconsistent values in **Item Fat Content**

2. Install the required libraries:
   ```
   pip install pandas numpy matplotlib seaborn
   ```
3. Open the Jupyter Notebook.
4. Run all cells to reproduce the analysis.

---

## 📌 Key Insights
- Low Fat products contribute more to total sales than Regular products.
- Fruits & Vegetables and Snack Foods are among the highest-selling item categories.
- Tier 3 outlets generate the highest sales.
- Medium-sized outlets contribute the largest share of sales.
- Sales vary across outlet establishment years and locations.
